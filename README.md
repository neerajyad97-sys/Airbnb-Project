# Airbnb_Global_Performance_Dashboard

## 1. Project Title
Airbnb Listing Performance & Customer Insights Dashboard | Power BI Business Intelligence Project

## 2. Short Description / Purpose
Developed an interactive Power BI dashboard to analyze Airbnb listings, customer ratings, review patterns, and property performance. The dashboard provides stakeholders with actionable insights to monitor listing success, understand customer satisfaction, identify improvement opportunities, and support data-driven business decisions.

## 3. Tech Stack
Power BI
* Power Query (Data Transformation)
* Data Modeling
* DAX (Data Analysis Expressions)
* Interactive Dashboard Development
* KPI Reporting
* Data Visualization

## 4. Features / Highlights

### Business Problems
* Lack of visibility into listing performance across multiple properties.
* Difficulty identifying factors affecting customer satisfaction.
* Large volumes of customer reviews making manual analysis inefficient.
* Challenges in monitoring property ratings and guest feedback trends.
* Limited data-driven insights for optimizing listing performance.
  
### Dashboard Goals
* Monitor overall Airbnb listing performance.
* Analyze customer ratings and review trends.
* Identify top-performing and underperforming properties.
* Track guest satisfaction metrics.
* Enable stakeholders to make informed business decisions through interactive reporting.
  
### Business Impact
* Improved visibility into property performance metrics.
* Enhanced understanding of customer satisfaction drivers.
* Helped identify areas requiring operational improvements.
* Reduced manual reporting efforts through automated visual analytics.
* Supported strategic decisions for improving occupancy and guest experience.
  
### Dashboard Highlights
* Executive Overview Dashboard with key performance indicators (KPIs).
* Ratings Analysis Dashboard for monitoring guest satisfaction.
* Reviews Dashboard for analyzing customer feedback trends.
* Interactive slicers and filters for customized analysis.
* Dynamic KPI cards for quick performance tracking.
* Drill-down capabilities for detailed property-level insights.
* Cross-filtering functionality for deeper data exploration.
* User-friendly and responsive dashboard design.
  
### Key Insights Delivered
* High-performing listings based on ratings and reviews.
* Customer satisfaction trends across properties.
* Review volume and engagement patterns.
* Areas requiring service and property improvements.
* Data-backed recommendations for listing optimization.
